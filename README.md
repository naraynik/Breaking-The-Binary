Breaking the Binary – SDG 5 Gender Equality Platform
Project Overview
Breaking the Binary is a web platform designed to support the United Nations Sustainable Development Goal 5 (SDG 5): Gender Equality. The website aims to empower individuals by providing access to educational opportunities, scholarships. It also highlights important statistics related to gender inequality and promotes awareness about the need for equal opportunities in education and career development.

The platform is designed using purple, the official color associated with SDG 5, symbolizing justice, dignity, and gender equality.

Purpose of the Project
Gender inequality continues to limit opportunities for many individuals, especially in education and career growth. Many students are unaware of available colleges, scholarships, and career paths that align with their interests and qualifications.

Breaking the Binary addresses this gap by providing a centralized platform where users can explore educational opportunities, understand gender equality challenges, and receive guidance on building their future.

Technologies Used
The project is developed using core web technologies including HTML, CSS3, and JavaScript(Vanilla JS). HTML is used to create the structure and layout of the website, organizing different pages and sections of the platform. CSS is used for styling and designing the user interface, ensuring a responsive and visually appealing experience. JavaScript is used to add interactivity and dynamic functionality to the platform, enabling features such as dashboard updates, displaying statistics, presenting college and scholarship information, and generating personalized guidance for users.
